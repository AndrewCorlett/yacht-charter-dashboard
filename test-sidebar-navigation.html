Sidebar Navigation Test

Test Instructions

The application is embedded below. Please verify the following functionality:

  • Sidebar shows both Dashboard and Admin Config menu items
  • Click on Admin Config switches to admin section
  • Dashboard menu item returns to main dashboard
  • Expand/collapse functionality works with both menu items
  • Active state highlighting works (blue background for active item)
  • Admin Config section shows placeholder content

Expected Behavior

Initial State: Sidebar should be collapsed (12px width) showing only icons. Dashboard should be the active section with blue highlighting.
Expand Sidebar: Click the chevron icon to expand sidebar to 64px width, revealing text labels "Dashboard" and "Admin Config".
Admin Config Navigation: Click "Admin Config" item - should highlight in blue and show admin placeholder content: "Admin Configuration" heading with "Admin configuration page coming soon..." text.
Dashboard Navigation: Click "Dashboard" item - should highlight in blue and return to main dashboard with SitRep section and calendar.
Collapse Sidebar: Click chevron again to collapse sidebar while maintaining active state highlighting.
